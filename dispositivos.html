Hardware del Proyecto

Dispositivos Utilizados

El sistema está compuesto por tres componentes principales: un microcontrolador ESP32, un sensor de temperatura y humedad DHT22, y un módulo de tarjeta SD para el almacenamiento local de los datos recolectados.

ESP32 C V4

Microcontrolador de doble núcleo con conectividad WiFi y Bluetooth integrada. Es el cerebro del sistema, encargado de leer el sensor, procesar los datos y almacenarlos en la tarjeta SD.

Especificaciones Técnicas

  • Procesador: Xtensa dual-core 32-bit LX6 hasta 240 MHz
  • Memoria ROM: 448 KB
  • Memoria SRAM: 520 KB
  • WiFi: 802.11 b/g/n — 2412 ~ 2484 MHz
  • Bluetooth: V4.2 BR/EDR y BLE
  • Flash: 4 MB SPI integrado
  • Voltaje de operación: 3.3V – 5V (GPIO a 3.3V)
  • Temperatura de operación: -40°C a 85°C
  • Periféricos: UART, SPI, I2C, I2S, GPIO, ADC, DAC, PWM, RTC, entre otros

Sensor DHT22 / AM2302

Sensor digital de temperatura y humedad de alta precisión. Permite registrar ambas variables ambientales con una sola conexión de datos al microcontrolador.

Especificaciones Técnicas

  • Voltaje de operación: 3.3V – 6V
  • Corriente de operación: 2.5 mA
  • Señal de salida: Digital — Bus de datos
  • Rango de humedad: 0% RH ~ 100% RH
  • Rango de temperatura: -40°C ~ 125°C
  • Precisión de humedad: ±2% ~ 5%
  • Velocidad de muestreo: 0.5 Hz (una lectura cada 2 segundos)
  • Temperatura de operación: -40°C ~ 80°C
  • Pines: VDD, DATA, NULL, GND

Módulo MicroSD Card Adapter

Módulo de almacenamiento que permite al ESP32 guardar los registros de temperatura y humedad directamente en una tarjeta microSD, asegurando la persistencia de los datos sin depender de internet.

Especificaciones Técnicas

  • Voltaje de operación: 4.5V – 5.5V
  • Voltaje interfaz SPI: 3.3V – 5V
  • Corriente de operación: 200 µA – 200 mA
  • Comunicación: SPI
  • Memoria soportada: uSD clásica hasta 2 GB
  • Memoria SDHC: Micro SDHC hasta 32 GB
