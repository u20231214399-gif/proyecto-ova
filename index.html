Descripción del Proyecto

Este proyecto consiste en el diseño de un sistema de monitoreo de temperatura dentro de un aula universitaria, con el objetivo de analizar el comportamiento de esta variable física en tiempo real.

Los datos recolectados permiten evaluar las condiciones ambientales del aula y comprender cómo influyen en el confort térmico y el rendimiento académico.

Ver Dispositivos

Importancia del Monitoreo

La temperatura es una variable física que impacta directamente el bienestar y la concentración de los estudiantes.

Monitorearla permite tomar decisiones informadas sobre ventilación, climatización y control ambiental dentro del aula.

Introducción

El monitoreo de variables físicas forma parte de las aplicaciones modernas del análisis de datos y sistemas embebidos.

En este OVA se presenta una propuesta básica de supervisión de temperatura ambiental mediante una plataforma web desarrollada en HTML5.
