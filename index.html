// Resumen Técnico

Sobre el Proyecto

El presente proyecto consiste en el diseño e implementación de un sistema de monitoreo de temperatura y humedad en un espacio interior residencial, utilizando un microcontrolador ESP32, un sensor DHT22 y un módulo de tarjeta SD para el almacenamiento local de los datos recolectados.

El sistema realiza lecturas periódicas de temperatura y humedad, registrando cada medición directamente en un archivo dentro de la tarjeta SD, lo que permite conservar el historial completo de datos sin depender de conexión a internet ni plataformas externas.

La información almacenada fue posteriormente analizada para evaluar el comportamiento térmico y de humedad del entorno, identificando patrones y variaciones a lo largo del tiempo de monitoreo.

Importancia del Monitoreo

Las condiciones térmicas y de humedad de un espacio interior influyen directamente en el bienestar y la salud de sus ocupantes. Niveles elevados de temperatura o humedad pueden generar incomodidad, fatiga y afectar la calidad del sueño y la concentración.

Contar con un sistema de registro continuo permite identificar momentos críticos del día, evaluar la ventilación del espacio y tomar decisiones para mejorar las condiciones ambientales del entorno residencial.

El uso de un módulo SD como medio de almacenamiento garantiza que ningún dato se pierda, incluso ante cortes de energía o reinicios del sistema, asegurando un registro confiable y completo.
